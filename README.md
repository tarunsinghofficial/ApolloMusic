# Apollo Music - A Spotify Clone
This app was built as a Spotify clone with some different features and UI.

## Features

* User Authentication
* Song share
* Song Download
* Mute option
* Radio enabled
* Offline music*
* Contact Us form

## Tech Stack Used

* React Native
* GraphQL API
* AWS Amplify
